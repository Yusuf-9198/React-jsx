# Beautiful Navbar Showcase

A modern, responsive navigation bar built with **React**, **Vite**, and **Tailwind CSS**.

## Features

✨ **Modern Design** - Clean and professional gradient navbar
📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
🎨 **Tailwind CSS** - Utility-first CSS framework for beautiful styling
⚡ **Vite** - Lightning-fast development experience
🔄 **Smooth Animations** - Hover effects and transitions
📦 **Lucide Icons** - Beautiful icon integration

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` to see your app running.

### Build

```bash
npm run build
```

## Navbar Features

- **Logo/Brand** - Customizable brand name
- **Desktop Navigation** - Clean navigation links
- **Mobile Menu** - Hamburger menu that collapses on mobile
- **CTA Button** - Sign In button for call-to-action
- **Smooth Animations** - Hover effects and transitions
- **Sticky Navigation** - Stays at the top while scrolling
- **Responsive Design** - Optimized for all screen sizes

## Customization

Edit `src/components/Navbar.jsx` to:
- Change brand name
- Modify navigation links
- Update colors (change `blue-600` and `purple-600` classes)
- Adjust button text
